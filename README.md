# Interactive DOM Manipulation Project

##  Project Title
Interactive DOM Manipulation Web Application

## 📖 Short Description
This project is a responsive interactive web application built using HTML, Tailwind CSS, and JavaScript DOM manipulation. It demonstrates real-time user interaction such as theme switching, dynamic lists, search filtering, dropdown menus, and interactive cards.

##  Features Implemented
- Light/Dark mode toggle
- Responsive navigation bar
- Dropdown menu (Services)
- Hamburger menu for mobile
- Dynamic activity cards
- Add & remove list items
- Search functionality
- Real-time text update
- Interactive buttons (Join Activity)

##  What the Project Demonstrates
This project demonstrates:
- DOM manipulation
- Event handling (click, keyup)
- Dynamic UI updates
- Responsive design
- State switching (dark/light mode)
- Interactive components using JavaScript
